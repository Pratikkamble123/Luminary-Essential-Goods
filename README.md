# 🛍️ Luminary – E-Commerce Platform

**Luminary** is a modern, full-featured e-commerce web application focused on clean design, smooth user experience, and practical shopping features. It is built to feel premium yet simple, with an interface tailored for everyday users.

The project demonstrates real-world e-commerce functionality while keeping the UI calm, trustworthy, and conversion-friendly.


## ✨ Features

* 🏠 **Modern Home Page** with hero section and collections
* 🛒 **Shopping Cart** with item quantity management
* 💳 **Checkout Flow** (cart → checkout → payment-ready)
* 📦 **Product Listing & Detail Pages**
* 🔍 **Search & Navigation** for easy browsing
* 🔐 **User-friendly UI** with clear actions
* ⚡ **Fast Performance** using Vite
* 📱 **Responsive Design** (desktop & mobile)


## 🧠 Project Goal

Luminary was built to showcase:

* A clean **frontend architecture** for an e-commerce app
* Practical **shopping flows** used in real products
* Calm, minimal UI suitable for Indian audiences
* A portfolio-ready full-stack style project

The focus is on clarity, usability, and real-world relevance rather than gimmicks.


## 🛠️ Tech Stack

* **Frontend:** React.js + TypeScript
* **Build Tool:** Vite
* **Styling:** CSS / modern utility styles
* **State Management:** Local & component state
* **Backend Ready:** Node.js (planned / extendable)
* **Database Ready:** PostgreSQL (planned / extendable)
* **Payments:** Stripe (integration-ready)

## 🚀 Getting Started

Follow these steps to run the project locally:

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/luminary.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd luminary
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

Open your browser at:

```
http://localhost:5173
```


## ⚙️ Environment Variables

Create a `.env.local` file in the root directory if required:

```
VITE_APP_NAME=Luminary
```

(Additional variables for payments or backend can be added later.)
